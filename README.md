# web3-js-vara-lib

该项目用于封装以下功能，供 flutter 端调用

- [] 根据助记词创建 vara 账户
- [] 计算 vara 创建交易的手续费
- [] 提供参数签署 vara 交易签名
- [] 广播 vara 交易

## 准备
```
yarn
yarn build
```

## 测试
```
cd test
cp env.example.js env.js
```
填充自己的配置到 env.js

开始测试:
```
node test
```

## webpack 配置

生成 flutter 可以直接使用的压缩文件
